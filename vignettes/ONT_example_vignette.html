1. Installation

This package depends on MEME and minimap2. MEME is a motif analysis tool which is used to determine the most common telomere motifs in a reference genome. Minimap2 is a read aligner which is used to align reads to the reference genome. Both of these tools are required for the package to run.

Install MEME

Download the software from https://meme-suite.org/doc/download.html Type the following commands:

tar -zxf meme-5.5.7.tar.gz
cd meme-5.5.7
./configure --prefix=$HOME/meme --enable-build-libxml2 --enable-build-libxslt
make
make test
make install

# Edit your shell configuration file to add $HOME/meme/bin and $HOME/meme/libexec/meme-5.5.7 to your shell's path. This can often be done by editing the file named .profile to add the following line:

export PATH=$HOME/meme/bin:$HOME/meme/libexec/meme-5.5.7:$PATH

Install minimap2

git clone https://github.com/lh3/minimap2
cd minimap2 && make

Process Fastq

This process is used in order to split files in order to support efficient parallelization. Ideally, you would set target_ram_size_gb * threads to be equal to something near the total RAM for your machine. The fastq file can be downloaded from the SRA database using SRA toolkit. Here are the commands you would use to download the fastq file for this experiment.

# Split large fastq file into files that will occupy
# 2GB of RAM in order to take advantage of all processors
# (20) with given RAM (62GB).
if (!file.exists(list.files(file.path(dirname(sample_file), "split"),
  full.names = TRUE
)[1])) {
  stats_fl <- auto_split(
    in_fastq = sample_file,
    prefix = sample_name,
    threads = 7,
    out_dir = file.path(data_dir, "split"),
    target_ram_size_gb = 2
  )

  # Save stats
  saveRDS(stats_fl, file.path(
    results_dir,
    "stats_fl.rds"
  ))
}

Find optimal thresholds

The method determines telomere boundaries using two threshold values for telomere frequency motifs in 200 bp rolling windows. The start threshold sets the minimum motif frequency required in the first window to identify a telomeric region, affecting the method’s sensitivity. The end threshold defines the minimum motif frequency that determines where the telomeric region ends - when a window falls below this threshold, that position marks the telomere length for that read. An optimal start threshold balances sensitivity, while the end threshold ensures accurate telomere length determination. The method’s performance hinges on these thresholds, as they directly influence the identification and measurement of telomeric regions in the sequencing data.
